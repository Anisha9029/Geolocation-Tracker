# Geolocation Tracker

## Overview
A simple Python application to find your public IP address and geolocation details such as city, region, country, ISP, and coordinates.  
It also generates a clickable Google Maps link for quick visualization. Perfect for learning API integration and IP-based location tracking.

## Features
- Retrieve your public IP address
- Get geolocation details:
  - City
  - Region
  - Country
  - Latitude & Longitude
  - Internet Service Provider (ISP)
- Generate a Google Maps link for your location
- Lightweight and easy to run in VS Code or any Python environment

## Requirements
- Python 3.x  
- `requests` library
